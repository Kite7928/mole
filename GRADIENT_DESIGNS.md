# 左右渐变设计方案

以下是为您设计的多种具有层次感的左右渐变方案，适用于侧边栏和其他UI元素。

## 方案一：经典绿紫渐变（当前使用）

**日间模式：**
```css
background: linear-gradient(180deg, rgba(7, 193, 96, 0.95) 0%, rgba(155, 140, 178, 0.95) 100%);
```

**夜间模式：**
```css
background: linear-gradient(180deg, rgba(76, 175, 80, 0.92) 0%, rgba(179, 157, 219, 0.92) 100%);
```

**特点：**
- 经典的微信绿到艺术紫
- 适合创作场景
- 层次感适中

---

## 使用方法

1. **选择方案**：从以上方案中选择一个您喜欢的
2. **修改侧边栏**：在 `frontend/components/layout/sidebar.tsx` 中替换 `aside` 标签的 `style` 属性
3. **修改Logo**：同时更新 Logo 图标的渐变
4. **修改用户头像**：更新用户头像的渐变
5. **修改移动端按钮**：更新移动端菜单按钮的渐变

## 推荐方案

- **创作场景**：方案一（经典）、方案三（深度）、方案六（波纹）
- **专业场景**：方案七（暗金）
- **创意场景**：方案八（彩虹）
- **技术场景**：方案十（霓虹）
- **极简风格**：方案九（极简）

## 颜色说明

- **绿色系**：微信绿 (#07c160)、Material绿 (#4caf50)
- **紫色系**：艺术紫 (#9b8cb2)、柔和紫 (#b39ddb)
- **蓝色系**：靛蓝 (#667eea)、天蓝 (#4299e1)
- **金色系**：暗金 (#daa520)、亮金 (#ffd700)
- **青色系**：青色 (#00f2fe)、青绿 (#00ffff)

您可以根据需要调整透明度和颜色组合！
